5.求欧拉函数

首先，欧拉函数是一个积性函数。

积性函数的定义是：函数$f(x)$,若满足$f(m n) = f(m) f(n) (gcd(n,m) = 1)$,那么$f(x)$就是一个积性函数。

6.一些神奇定理

  • $\sum_{d|n}\varphi(d) = n$
  • $\forall n > 1,1-n中与n互质的数的和为n \times \varphi(n) / 2$

  • $若gcd(a,n) = 1 \ \ \ \ \forall 正整数b \ \ \ \ a^b \equiv a^{b \ mod \ \varphi(n)}(mod\ n)$

  • 若$a$,$n$互质，则满足$a^x \equiv 1 (mod \ n)$的最小正整数$x_0$是$\varphi(n)$的约数。
